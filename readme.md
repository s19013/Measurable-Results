ビジネスとか転職とかでは、数字で語れる成果が大事である。

# 例
## 営業職の場合
* 営業経験10年の内、年間目標達成は9回
* 部下や後輩の商談同行件数は年間100件以上
  * -> 「育成」や「チームワーク」の意識の高さ

## アシスタント業務の場合
* 議事録の累積作成件数は200件以上かつ、翌日までの送付率は90%
* 〇〇の申請方法をレクチャーした相手は100名以上
* 作成したマニュアルは10種類以上

# 給与計算が主担当の労務
業務プロセスのデジタル化プロジェクトを立ち上げ、毎月の給与計算にかかる業務時間を半年後に10時間効率化

# カスタマーサポートの場合
問い合わせ対応後に送信するアンケート回答率3%、
回答結果の満足度top2スコアで80%実現

# 業務改善or新規事業の起案の場合
5年で10件の起案を行い、1つの部門で導入
-> 「改善志向」や「主体性」の高さ
